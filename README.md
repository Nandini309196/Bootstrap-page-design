# Bootstrap-page-design
This project is a responsive Bootstrap 5 website created to demonstrate core web development concepts including HTML, CSS, JavaScript, and Bootstrap.
It includes three main pages:

Home Page (index.html) – Introduces the website and highlights features.

About Page (about.html) – Shares details about the team and project goals.

Contact Page (contact.html) – Provides a user-friendly contact form for feedback.

The project emphasizes modern design, responsiveness, and usability, using Bootstrap’s grid system, custom CSS styling, and media assets for a professional finish.
